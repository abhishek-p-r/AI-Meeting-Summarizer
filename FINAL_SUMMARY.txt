║ HOW TO USE                                                                 ║
╚════════════════════════════════════════════════════════════════════════════╝

QUICK START (One Command):
───────────────────────────────────────────────────────────────────────────
$ chmod +x run_meeting_summarizer.sh
$ ./run_meeting_summarizer.sh

Then open: http://localhost:7860


MANUAL START:
───────────────────────────────────────────────────────────────────────────
# Terminal 1: Ollama server
$ ollama serve

# Terminal 2: App
$ python3 -m venv .venv
$ source .venv/bin/activate  # Windows: .venv\Scripts\activate
$ pip install -r requirements.txt
$ python main.py

Then open: http://localhost:7860


USAGE:
───────────────────────────────────────────────────────────────────────────
1. Upload audio file (MP3, WAV, M4A)
2. Add optional meeting context
3. Select Whisper model (small/medium/large)
4. Select summarization model (llama2, mistral, etc)
5. Click "▶️ Generate Summary"
6. View results and download transcript
